# FirstVet
    Algorithm which is finding suitable appointments for veterinarian
# Set up 
- Run local/dev server
    - yarn && yarn start:dev

- Generate production build 
    - yarn build

- Run production server
    - yarn start:prod

# Run unit tests
    - yarn test
    - To watch all test
        - yarn test:watch
    - To watch uni tests coverage
        - yarn test:coverage
